# TSWApi

F# library for interacting with the Train Sim World 6 API.

## Getting Started

See the [Quickstart Guide](quickstart.html) for setup instructions.

## Features

- **Type-safe API client** — Strongly typed F# wrappers for the TSW6 REST API
- **Automatic auth** — Discovers and loads the DTGCommKey from your local game installation
- **Tree navigation** — Explore the TSW node hierarchy programmatically
- **Async-first** — All operations return `Async<Result<'T, ApiError>>`
